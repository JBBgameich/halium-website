Introduction

Halium is a project that aims to unify the hardware abstraction layers (HAL) of different projects targeting mobile devices shipped with android. Halium will be a base for Linux distributions for phones and tablets, that enables them using sensors, camera and hardware-accelerated graphics. Halium includes libhybris, a phone specific kernel from android and a phone specific android rootfs, where halium get's the needed drivers from.

The project is still in an early stage, but you can follow our project at Github. Documentation is available on Github.com/Halium/docs.
